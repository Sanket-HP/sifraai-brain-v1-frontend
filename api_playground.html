SIFRA NEURAL PLAYGROUND

An advanced, low-latency interface for interacting with Google's generative AI models. Design, test, and execute API payloads in real-time.

01. INITIALIZATION

  1. Locate the Authorization Key field in the top-left panel.
  2. Enter your Google Gemini API Key. (Stored locally in browser for convenience)
  3. Select a Neural Model from the dropdown menu (e.g., Gemini 2.5 Flash).
  4. Type a request in the Prompt / Input box.
  5. Click INITIALIZE SEQUENCE to execute.

02. MODEL CAPABILITIES

GEMINI 2.5 FLASH (TEXT)

High-speed text generation. Good for chat, coding, summarization, and extraction tasks.

IMAGEN 4.0 (IMAGE GEN)

State-of-the-art text-to-image generation. Describe a scene in detail for best results.

GEMINI 2.5 FLASH (IMAGE EDIT/VISION)

Multimodal model. Can analyze images (Visual QA) or edit them based on instructions. Requires Base64 image data.

GEMINI 2.5 FLASH (TTS)

Text-to-Speech synthesis. Converts your prompt into high-quality audio waveforms.

03. UTILITY MODULES

Image Encoder

Click the IMG ENCODER button in the top navigation bar.

  • Drag & Drop an image file.
  • The tool converts it to a clean Base64 string.
  • Click COPY and paste it into the "data" field of your JSON payload for Image Editing requests.

JSON Editor

The Payload.json editor is live. While the Quick Prompt box updates the text field, you can manually edit parameters like temperature, topK, or systemInstruction directly in the JSON for advanced control.

04. DIAGNOSTICS

  • 400 Bad Request: Check your JSON syntax. Ensure quotes are closed and commas are correctly placed.
  • 401 Unauthorized: Your API Key is missing or invalid. Re-enter it in the config panel.
  • No Response: If the loading spinner persists, check your internet connection or the API status.
