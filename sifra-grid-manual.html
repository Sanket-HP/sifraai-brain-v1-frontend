CORE_ARCHITECTURE

SIFRA GRID is a deterministic, client-side execution environment built for high-stakes data manipulation. Unlike traditional cloud spreadsheets, all processing happens locally within your browser's V8 engine.

THE MATRIX

Structured tabular format with row indexing and column labeling (A, B, C...). Includes live focus reveals for hidden formulas.

SYSTEM CONSOLE

Real-time telemetry log tracking every operation. Critical for verifying data commits and export readiness.

FORMULA_ENGINE

The deterministic engine supports standard spreadsheet syntax. Every formula must be prefixed with the = operator to trigger the parser.

ARITHMETIC_OPS
Addition: =A1+B1
Sub: =A1-B1
COMPLEX_OPS
Mult: =C1*D5
Div: =E1/F1

Cell Referencing Protocol

Columns map to letters (A, B, C...) based on sequence. Rows map to indices (1, 2, 3...).

Example: =B2*1.15
Multiplies the value in column B, row 2 by a 15% markup factor.

DATA_INJECTIONS

Column Injection

  1. Activate the Inject_Column command in the Sidebar.
  2. Define a unique Header Identifier.
  3. The system appends the column with the next available alpha-label.

HEALTH_&_SECURITY

Storage Warning

Grid memory is VOLATILE. Refreshing the browser without a SAVE_STATE dump will result in total data purge.

Data Sovereignty

All logic is executed locally. Zero telemetry packets containing user data are transmitted to external nodes.
